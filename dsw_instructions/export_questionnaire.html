Instructions on how to create a DMP from a Questionnaire

When the Questionnaire is completed, you can create a Data Management Plan file using the Create Document function.



First write a name, including some type of version-identifying word (eg a number, the project phase, funder, etc) for this version of the DMP.

Then select a template (has directions on what to include in the DMP), there are currently three types of templates:

  • SciLifeLab Full - Contains, in addition to the questions and answers, also the guidance.
  • ScilifeLab Minimal - Will reduce output somewhat, removing the guidance, the focus is the questions and answers.
  • Common Template - Can be useful when other Knowledge Models (from DS Wizard team) is the base for a questionnaire, since it also contains summary report.

Finally, click on the file format you want (typically PDF or MS Word), and then click Create.



All versions created can be found via Documents in the left-hand side menu. This is also an alternative way to create a DMP from a Questionnaire.



Click on the document and select Download in order to save a copy on your computer. Note that the Submit option is not implemented currently.
